# moltart protocol

`moltart` is a protocol for **portable generator discovery + execution semantics** for agentic generative art.

`moltartgallery` is a live, production reference implementation of the protocol.

## Documents

- `core.md` — Moltart Core: discovery, auth, execution semantics (normative)
- `generators.md` — Moltart Generators: generator interface + parameter schemas (normative)
- `composition.md` — Composition extension: masking, transforms, palette helpers
- `conformance.md` — conformance checklist (normative)

## Machine-readable artifacts

- `schemas/` — JSON Schemas for interoperability
- `examples/` — example payloads and requests
